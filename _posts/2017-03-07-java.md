## 반복문에서 break를 만나면 반복문을 빠져나간다.
```
while(true) {
    sum += i;
    if( i == num ) break;    //i값이 사용자가 입력한 값과 같으면 반복을 벗어남
    i++;
}
```

## 반복문에서 continue를 만나면 반복문의 처음으로 돌아간다.
```
for(int i = 1; i<=num; i++) {
    if(i % 2 == 1) continue;    //i값이 홀수값이면 더하지 않고 반복문의 처음으로 이동
    sum += i;
}
```
